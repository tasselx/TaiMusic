# Tai Music

酷狗音乐 Tauri 客户端，基于 [KuGouMusicApi](https://github.com/MakcRe/KuGouMusicApi) 开发。

## 功能特点

- 使用Tauri构建的轻量级桌面应用
- 集成了酷狗音乐API的完整功能
- 可搜索音乐、查看轮播图等内容
- 跨平台支持（Windows、macOS、Linux）

## 开发

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run tauri dev
```

### 构建应用

```bash
npm run tauri build
```

## 技术栈

- Tauri + React + TypeScript
- 后端API基于Node.js + Express
